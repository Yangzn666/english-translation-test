## ✨ 新增功能

### 1. 题目分组功能 📋

#### 功能描述
将所有题目按每组15题进行分组，用户可以自由选择做哪一组，无需每次都从头开始。

#### 使用方法
1. 选择练习模式（词组/词汇）后，会出现分组选择区域
2. 点击"全部"按钮：练习所有题目
3. 点击"第X组"按钮：只练习该组的15道题目
4. 分组选择会自动保存，下次进入时恢复

#### 分组示例
- **词组模式**（140题）：
  - 全部：140题
  - 第1组：1-15题
  - 第2组：16-30题
  - ...
  - 第10组：136-140题（最后一组可能不足15题）

- **词汇模式**（110题）：
  - 全部：110题
  - 第1组：1-15题
  - 第2组：16-30题
  - ...
  - 第8组：106-110题

#### 优势
✅ **灵活选择**：可以针对性地练习薄弱部分  
✅ **节省时间**：不必每次都从第1题开始  
✅ **进度管理**：每组独立统计，更容易掌握进度  
✅ **自动保存**：分组选择会保存到localStorage

### 2. 二级提示功能 🔍

#### 功能描述
在原有的一级提示基础上，增加更详细的二级提示，帮助用户更好地回忆答案。

#### 提示级别

**一级提示（原提示）**
- 词组模式：显示首字母、总长度、单词数
- 词汇模式：显示遮蔽形式的单词（如 `i_t_r_s_`）、字母数、首字母

**二级提示（新增）**
- 词组模式：显示50%的字母，其余用下划线代替
- 词汇模式：显示60%的字母，其余用下划线代替

#### 使用方法
1. 点击"💡 显示提示"按钮 → 显示一级提示
2. 如果还不够，点击"🔍 显示更详细提示（二级）"按钮
3. 二级提示会显示更多字母，帮助回忆

#### 示例

**词汇模式 - interest**
- 一级提示：`i_t_r_s_` (7个字母，首字母: I)
- 二级提示：`i_nt__e_t` （显示了约60%的字母）

**词组模式 - take advantage of**
- 一级提示：首字母 T, 17个字符，3个单词
- 二级提示：`t_k_ a_v__t_g_ o_` （显示了约50%的字母）

#### 优势
✅ **渐进式提示**：从简到繁，给用户思考空间  
✅ **更详细线索**：二级提示提供更多字母信息  
✅ **灵活控制**：用户可以选择是否需要二级提示  
✅ **记忆激活**：通过部分字母激活完整记忆

### 分组功能

```javascript
// 状态变量
let selectedGroup = 'all'; // 'all' 或组号 (0, 1, 2...)
const GROUP_SIZE = 15; // 每组题目数量
```

### 二级提示
```javascript
// 核心函数
showLevel2Hint()          // 显示二级提示
revealMoreLetters(word, ratio)  // 按比例显示字母
```


## 🎯 使用建议

### 场景1：系统性学习
1. 选择"全部"模式
2. 从头到尾完成所有题目
3. 利用一级和二级提示辅助记忆
4. 记录错题，重点复习

### 场景2：针对性练习
1. 根据薄弱环节选择特定分组
2. 例如：只练习第3-5组（形近词较多的部分）
3. 反复练习直到熟练掌握
4. 再切换到其他分组

### 场景3：快速复习
1. 选择最近的分组
2. 快速过一遍题目
3. 不会的直接看答案
4. 标记难点，后续重点练习

### 场景4：碎片时间学习
1. 选择一组（15题）
2. 利用5-10分钟完成
3. 随时中断，进度自动保存
4. 下次继续同一分组

## 💡 最佳实践

### 提示使用策略
1. **先尝试自己回答**：不要立即看提示
2. **一级提示不够再用二级**：给大脑思考的机会
3. **二级提示还不会再看答案**：避免过度依赖
4. **记录使用提示的题目**：这些是需要重点记忆的

### 分组学习策略
1. **新内容**：选择"全部"模式，系统学习
2. **复习**：选择特定分组，针对性强化
3. **测试**：随机选择分组，检验掌握程度
4. **冲刺**：重点练习错误率高的分组
