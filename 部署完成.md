## ❓ 常见问题

### Q1: 页面显示 404？
**A**: 等待 1-3 分钟，GitHub Pages 需要时间部署。刷新页面再试。

### Q2: 样式加载失败？
**A**: 检查网络连接，Tailwind CSS 通过 CDN 加载。尝试硬刷新（Ctrl + F5）。

### Q3: 如何自定义域名？
**A**: 在 GitHub Pages 设置中添加自定义域名，并配置 DNS CNAME 记录。

### Q4: 可以设为私有仓库吗？
**A**: GitHub Pages 免费用户只能公开仓库。私有仓库需要 GitHub Pro。
