# 英语翻译检测小程序

## 📖 功能介绍

这是一个专为英语一翻译学习设计的检测小程序，包含两个核心功能模块：

### 1️⃣ 词组检测
- **显示中文含义**，用户需要拼写对应的英文词组
- **智能判分**：自动忽略空格和大小写差异
- 例如：输入 "take advantage of" 或 "TakeAdvantageOf" 都会被判定为正确

### 2️⃣ 词汇检测
- **显示中文含义**，用户需要拼写对应的英文单词
- **多答案支持**：如果一个中文对应多个英文单词，需要全部拼写
- 例如："强调" 对应 ["emphasize", "stress", "highlight"]，需要输入所有三个单词
- 多个答案之间可以用逗号、顿号或空格分隔

## 🎯 主要特性

✅ **实时统计**：显示总题数、正确数、错误数和正确率  
✅ **进度追踪**：可视化进度条，清晰了解学习进度  
✅ **即时反馈**：答对显示绿色提示，答错显示红色提示并给出正确答案  
✅ **查看答案**：遇到困难时可以查看正确答案  
✅ **键盘快捷**：按 Enter 键快速提交答案和进入下一题  
✅ **响应式设计**：完美适配手机、平板和电脑  

## 🚀 使用方法

### 方式一：直接打开（推荐）
1. 双击 `index.html` 文件
2. 浏览器会自动打开小程序
3. 开始练习！

### 方式二：使用本地服务器
```bash
# 如果使用 Python
cd "d:\学习\效率\02-英语一\01-知识点整理\翻译\翻译检测小程序"
python -m http.server 8080

# 然后在浏览器访问 http://localhost:8080
```

## 📝 操作步骤

1. **选择模式**：点击顶部的"词组检测"或"词汇检测"按钮
2. **开始练习**：点击绿色的"开始练习"按钮
3. **输入答案**：在输入框中输入英文
4. **检查答案**：点击"检查答案"按钮或按 Enter 键
5. **查看反馈**：系统会显示正确或错误的提示
6. **继续下一题**：点击"下一题"按钮或按 Enter 键
7. **完成练习**：所有题目完成后会显示统计结果

## 💡 使用技巧

### 词组检测技巧
- 不需要担心空格：输入 "takeadvantageof" 和 "take advantage of" 效果相同
- 不需要担心大小写：输入 "TAKE ADVANTAGE OF" 也能通过
- 建议正常书写，养成良好的拼写习惯

### 词汇检测技巧
- 多答案题目会有黄色提示框提醒
- 多个答案之间可以用以下方式分隔：
  - 逗号：`emphasize, stress, highlight`
  - 顿号：`emphasize、stress、highlight`
  - 空格：`emphasize stress highlight`
- 必须输入所有正确答案才算对

## 📊 数据统计

练习过程中会实时显示：
- **总题数**：当前模式的题目总数
- **正确**：答对的题目数量
- **错误**：答错的题目数量
- **正确率**：正确题目占总答题数的百分比
- **进度条**：直观显示当前练习进度

## 🔄 重新开始

完成练习后，可以点击弹窗中的"重新开始"按钮，或者刷新页面重新开始练习。
