# 🚀 快速部署指南

## 只需 3 步，即可获得独立网址！

---

## 第 1 步：创建 GitHub 仓库

1. 访问 https://github.com/new
2. 填写信息：
   - **Repository name**: `english-translation-test`
   - **Description**: `英语翻译检测小程序`
   - **Visibility**: ✅ **Public**（必须公开）
   - ❌ 不要初始化 README
3. 点击 **Create repository**
4. 复制仓库地址（类似）：
   ```
   https://github.com/您的用户名/english-translation-test.git
   ```

---

## 第 2 步：推送代码

### 方法 A：使用自动化脚本（推荐）

双击运行：**`推送到GitHub.bat`**

按提示输入您的仓库地址即可！

### 方法 B：手动命令行

```bash
cd "d:\学习\效率\英语翻译检测小程序"

# 替换为您的实际仓库地址
git remote add origin https://github.com/您的用户名/english-translation-test.git

git branch -M main
git push -u origin main
```

---

## 第 3 步：启用 GitHub Pages

1. 在 GitHub 打开您的仓库
2. 点击 **Settings** → **Pages**
3. 配置：
   - Source: **Deploy from a branch**
   - Branch: **main** / **(root)**
4. 点击 **Save**
5. 等待 1-3 分钟 ⏳

---

## 🎉 完成！

您的独立网址将是：
```
https://您的用户名.github.io/english-translation-test/
```

### 特性
✅ 完全独立于效率网站  
✅ 不影响主项目运行  
✅ 支持移动端访问  
✅ 免费托管  
✅ 自动 HTTPS

## 🔄 后续更新

修改代码后，只需：
```bash
git add .
git commit -m "更新说明"
git push
```

GitHub Pages 会自动重新部署（约 1-3 分钟）。
