# 英语翻译检测小程序 - 更新说明

## 📊 数据更新

### 词组检测
- **数据来源**：中英翻译练习.html
- **题目数量**：140题（完整版）
- **涵盖范围**：
  - 基础动词短语（take advantage of, give rise to等）
  - 常用表达（in private, on purpose等）
  - 翻译常用表达（make a living, make it等）
  - 工业化城市化相关词汇（industrialization, urbanization等）
  - 环境保护相关词汇（environmental protection, global warming等）

### 词汇检测
- **数据来源**：学习笔记系统.html
- **题目数量**：202题（完整版，包含形近词对比）
- **涵盖范围**：
  - 普通词汇（interest, field, society, despite等）
  - 常用词组（sure enough, pay attention to, broaden horizon等）
  - 形近词对比（private/pirate, crow/crown/clown, doze/dozen等161组词）

## ✨ 新增功能

### 1. 题号显示
- 在统计面板中新增"当前题号"显示
- 实时显示当前正在回答的题目序号
- 方便追踪学习进度

### 2. 断点续练功能
- **自动保存**：每次答题后自动保存当前进度到浏览器本地存储
- **自动恢复**：下次打开时自动恢复到上次的题目位置
- **独立保存**：词组和词汇的进度分别保存，互不干扰
- **清除进度**：新增"清除进度"按钮，可随时重置学习进度

### 3. 进度条优化
- 将"清除进度"按钮整合到进度条区域
- 更清晰的布局，方便操作

## 🎯 使用说明

### 开始练习
1. 双击 `启动.bat` 或 `打开程序.vbs` 打开程序
2. 选择"词组检测"或"词汇检测"模式
3. 点击"开始练习"按钮

### 答题方式
- **词组检测**：看到中文，拼写英文词组（可忽略空格和大小写）
- **词汇检测**：看到中文，拼写英文单词（如有多个答案需用逗号或空格分隔）

### 断点续练
- 程序会自动保存您的学习进度
- 关闭浏览器后再打开，会从上次中断的题目继续
- 如需重新开始，点击"清除进度"按钮

### 快捷键
- **Enter键**：快速提交答案/进入下一题

## 🔧 技术实现

- 使用 localStorage API 实现数据持久化
- 支持跨会话保存和恢复学习进度
- 响应式设计，支持手机和电脑访问
- 纯前端实现，无需服务器

## 💡 提示

1. 建议使用 Chrome、Edge 或其他现代浏览器
2. 定期清除浏览器缓存可能会影响进度保存
